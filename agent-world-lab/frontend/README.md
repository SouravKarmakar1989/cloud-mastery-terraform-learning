# Frontend

React + TypeScript frontend for Agent World Lab.

Implemented first-pass features:
- Login view
- Chat view
- Session/thread panel
- Approval panel
- Tool usage panel
- Trace and audit panel
- Environment and provider selectors

## Local run

1. Ensure backend API is running on `http://localhost:5000`.
2. Copy `.env.example` to `.env` and adjust if needed.
3. Run:

```bash
npm install
npm run dev
```

## Build and test

```bash
npm run build
npm run test
```
