# Common Environment Settings

This folder provides shared variable examples used across the split cloud stacks.

## Stack locations

- `azure-infrastructure/ai-foundry/genAI-agentic/agent-world-lab-stack`
- `aws-infrastructure/ai/bedrock/agent-world-lab-stack`
- `gcp-infrastructure/ai/vertex-ai/agent-world-lab-stack`

## Shared variable model

- `project_name`
- `platform_short`
- `environment`
- `owner`
- `cost_center`
- `data_class`
- `feature_flags` object

## Feature flags

- `enable_azure_foundry`
- `enable_bedrock_agentcore`
- `enable_gcp_advanced`
- `enable_vector_store`
- `enable_observability_stack`
