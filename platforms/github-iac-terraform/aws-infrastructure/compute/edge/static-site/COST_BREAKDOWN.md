# Cost Breakdown: edge/static-site

## Components
- S3 storage and request costs.
- CloudFront data transfer out and request charges.
- Optional invalidation requests and TLS/custom domain extras.

## Pre-Deploy Cost Checklist
1. Estimate static content size and monthly growth.
2. Estimate CDN request volume and egress by region.
3. Estimate cache hit ratio and invalidation frequency.
