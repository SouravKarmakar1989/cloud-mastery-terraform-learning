# Cost Breakdown: paas/elastic-beanstalk

## Components
- Underlying EC2, ASG, and ELB resources.
- EBS volumes and data transfer.
- Optional RDS/ElastiCache resources (if attached).

## Pre-Deploy Cost Checklist
1. Choose environment tier and capacity profile.
2. Estimate baseline and peak instance counts.
3. Include load balancer and storage costs.
