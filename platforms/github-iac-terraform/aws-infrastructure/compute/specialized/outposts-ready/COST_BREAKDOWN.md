# Cost Breakdown: specialized/outposts-ready

## Components
- SSM parameters (minimal cost footprint).
- If activated for Outposts: hardware capacity subscription and on-prem rack services.
- Data transfer between Outposts and regions.

## Pre-Deploy Cost Checklist
1. Validate whether Outposts is in-scope for environment.
2. If yes, include rack capacity and support pricing model.
3. Estimate regional data synchronization traffic.
