# Cost Breakdown: specialized/hpc-primitives

## Components
- Placement groups: no direct charge.
- Associated HPC compute nodes (EC2), EBS, high-throughput networking.
- Optional FSx for Lustre / EFA / parallel file systems (if added).

## Pre-Deploy Cost Checklist
1. Estimate HPC node-hours by instance family.
2. Estimate storage and network throughput requirements.
3. Include job scheduler and observability stack costs.
