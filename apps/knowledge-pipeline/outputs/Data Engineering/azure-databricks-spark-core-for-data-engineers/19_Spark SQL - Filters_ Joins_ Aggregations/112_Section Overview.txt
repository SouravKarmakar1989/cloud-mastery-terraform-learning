Welcome to a new section of the course. Now that we've created Spark tables for all of our data, it will be the right time to learn about how to use SQL to read and transform the data. In this section, we'll take a similar approach to the way we learned about Spark DataFrames. First, we'll look at how to read or select data from Spark SQL tables, apply filters, order the data being selected, etc.. This will be an introductory lesson, so please feel free to escape that, if you're proficient with SQL. We'll then look at simple functions that you can apply on the data being selected, for example, concatenating two fields to create one. Will then look at doing aggregations by grouping data. We'll then learn about performing functions such as Rank, Lead, Lag, etc., on a window of data which are called Window Functions. Finally, we'll learn about different types joints available and look at some examples of joining the data in Spark SQL. By the end of this section, I hope you will have gained a good understanding about how to read and manipulate the data in Spark SQL to gain insights. Let's get started.
