The first one is the load balancer access logs to get details of all the requests meant to your load balancers. And so it's for the ALB, the NLB and the CLB. And you can export these lugs into S3. CloudTrail Logs which are the logs of all the API calls met within your accounts, and this can be exported to S3 and CloudWatch Logs. VPC Flow Logs, which is going to give you information about all the IP traffic going to and from your network interfaces, in your VPC. And this can be sent to S3, to CloudWatch Logs and to Kinesis Data Firehose. You have Route 53 Access Logs and these can be sent and streamed directly into CloudWatch Logs and they will log information about all the queries that Route 53 is receiving. S3 Access Logs where you can send this to another S3 bucket which is giving you some information about all these access logins that happens for the request made within your bucket. CloudFront Access Logs to S3 which is going to give you detailed information about every user request that CloudFront will receive. And finally, all the config information can be exported into S3, so you can get all your configuration exported as a backup and do some analysis on them.
