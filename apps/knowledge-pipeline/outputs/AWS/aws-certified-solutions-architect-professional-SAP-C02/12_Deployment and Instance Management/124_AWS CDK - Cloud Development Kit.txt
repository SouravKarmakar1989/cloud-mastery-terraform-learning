Therefore, you can deploy your infrastructure and your application runtime code together because they can possibly share the same languages. Which is great for Lambda function, great for Docker containers in ECS and EKS. So let's take an example. So let's choose Python, for example, as a programming language. But the idea is that we want to use a cloud infrastructure using a programming language, because it allows us to get, for example, type safety.
