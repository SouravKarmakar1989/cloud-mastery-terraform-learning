# Amazon FSx - Hands On

## Pointwise No-Loss Summary

1. Demo reviews FSx create-flow and compares major file-system options in console.

## File System Options Walkthrough

1. FSx for Lustre:
   - deployment/storage choices,
   - throughput settings,
   - VPC/encryption options.
2. FSx for Windows File Server:
   - Single-AZ vs Multi-AZ,
   - SSD/HDD,
   - capacity/throughput coupling,
   - Active Directory integration options,
   - encryption and operational settings.
3. FSx for NetApp ONTAP:
   - deployment choices,
   - storage capacity/type,
   - storage-efficiency features (dedupe/compression/compaction context).
4. FSx for OpenZFS:
   - managed ZFS setup options and compatibility context.

## Outcome

1. Hands-on focuses on recognizing provisioning choices and mapping each FSx family to intended workload type rather than deep operational execution.
