So there is a metric called DDoSDetected which indicates whether there's a DDoS event happening for a specific resource. There is a DDoSAttackBitsPerSecond, which gives you the number of bits attacking a specific resource. There is the DDoSAttackPacketsPerSecond which gives you the number of packets being sent for a specific resource. And then the DDoSAttackRequestsPerSecond, which gives you an idea of how many requests are happening on a specific resource. And all these things combined is, basically, allowing you to understand what is happening and how the attack is being conducted.
