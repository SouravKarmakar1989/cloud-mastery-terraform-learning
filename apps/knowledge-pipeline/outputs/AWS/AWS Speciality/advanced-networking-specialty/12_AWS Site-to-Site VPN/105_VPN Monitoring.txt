So, you can do VPN monitoring using CloudWatch because AWS Site-to-Site VPN is a native service by AWS. It uses virtual private gateway. You can very well get some metrics from this virtual private gateway using CloudWatch and then you can do all the regular stuff that you typically do. For example, you can create alarms and then you can get notified when something goes wrong. So, in particular, AWS Site-to-Site VPN provides three important CloudWatch metrics. Very first is TunnelState. So, the state of the tunnel, which indicates whether tunnel is down or it's up. So, zero indicates that tunnel is down and one indicates that tunnel is up. Now this TunnelState could also have different values, anything between zero and one. And in those cases it would indicate that at least one of the tunnel is not up. So, zero means both the tunnels are down. One means both tunnels are up. But if it's between zero and one that may mean there is one tunnel which is not up. So, that's a TunnelState CloudWatch metric. Then there is TunnelDataIn, the bytes that through the VPN tunnel. And then TunnelDataOut, the total bytes sent through the VPN tunnel. All these metrics are collected by default using CloudWatch. You don't have to do any additional settings for this. And as you have these metrics available you can definitely create a CloudWatch Alarm. For example, when TunnelState goes down that means value becomes zero, you can say: set an alarm, send a notification via AWS notification service, that is SNS, and you can send either email and SMS to the receiver or administrator. So, remember these three CloudWatch metrics: TunnelState, TunnelDataIn and TunnelDataOut. Those are available for you to monitor via CloudWatch. Now, there is one more way in which you can do the monitoring of your VPN tunnel, and that's through a Health Dashboard. Now, as you know, AWS provides something called Personal Health Dashboard, PHD and any changes to the VPN tunnel you can see that automatically coming to your Health Dashboard. So, AWS Site-to-Site VPN automatically sends the notification to AWS Personal Health Dashboard.
