# Demo - Creating Your First Bucket

## Pointwise No-Loss Summary

1. Demo creates first S3 bucket in console and reviews key creation options.
2. Emphasis includes naming, region choice, public-access block, versioning, and encryption defaults.

## Bucket Creation Flow

1. Open S3 -> Buckets -> Create bucket.
2. Provide globally unique bucket name.
3. Select target region.
4. Object ownership left at default bucket-owner enforced setting.
5. Block Public Access kept enabled.
6. Bucket versioning enabled in demo.
7. Optional tagging available.
8. Default server-side encryption configured (SSE-S3 default shown; KMS option discussed).
9. Bucket key option reviewed.
10. Object Lock advanced setting shown but not used in this step.
11. Create bucket.

## Outcome

1. Bucket is created successfully and appears in account inventory.
2. Demonstrates immediate readiness for object uploads.
