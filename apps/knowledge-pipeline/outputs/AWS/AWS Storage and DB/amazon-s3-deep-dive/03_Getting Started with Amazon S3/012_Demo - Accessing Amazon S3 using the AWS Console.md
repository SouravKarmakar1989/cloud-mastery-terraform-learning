# Demo - Accessing Amazon S3 using the AWS Console

## Pointwise No-Loss Summary

1. Demo shows multiple ways to navigate to S3 service in console.
2. Region selection is highlighted before opening S3.

## Access Paths Demonstrated

1. Use top search bar and type S3.
2. Use Services menu -> Storage category -> S3.
3. Use common-service shortcut tiles when present.

## S3 Console Orientation

1. Open Buckets view (new account shows no buckets initially).
2. Left-side menu includes key S3 features such as:
   - buckets,
   - access points,
   - batch operations,
   - block public access,
   - storage lens,
   - additional S3 controls.

## Outcome

1. User reaches S3 console and is ready to create/manage first bucket.
