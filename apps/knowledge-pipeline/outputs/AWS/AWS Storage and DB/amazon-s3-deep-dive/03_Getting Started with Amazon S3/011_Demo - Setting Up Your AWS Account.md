# Demo - Setting Up Your AWS Account

## Pointwise No-Loss Summary

1. Demo shows creating a new AWS account from aws.amazon.com signup flow.
2. Process includes identity verification, billing setup, and support-plan selection.

## Account Creation Steps Shown

1. Open AWS signup entry point.
2. Enter root-user email and account name.
3. Verify email via code.
4. Create root-user password.
5. Enter contact/profile information.
6. Add billing method (card details).
7. Complete phone/SMS identity verification.
8. Select support plan (basic free in demo).
9. Complete signup and access AWS Management Console.

## Post-Signup Note

1. Instructor notes root account should not be used as daily admin identity.
2. Best practice is to create separate users/roles for routine management.

## Outcome

1. New AWS account is successfully created and console login is confirmed.
