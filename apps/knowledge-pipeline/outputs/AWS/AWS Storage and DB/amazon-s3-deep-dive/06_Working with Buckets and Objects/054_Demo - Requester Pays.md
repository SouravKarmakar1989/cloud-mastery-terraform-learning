# Demo - Requester Pays

## Pointwise No-Loss Summary

1. Demo shows where to enable/disable Requester Pays in S3 bucket settings.
2. Configuration path is bucket-level in Properties tab.

## Steps Demonstrated

1. Open target bucket.
2. Go to Properties.
3. Scroll to Requester Pays setting.
4. Click Edit.
5. Enable Requester Pays and save.

## Practical Meaning Reinforced

1. Owner continues paying for storage.
2. External requesting accounts are billed for request/transfer activity against the bucket.

## Outcome

1. Learner can locate and configure Requester Pays in console.
