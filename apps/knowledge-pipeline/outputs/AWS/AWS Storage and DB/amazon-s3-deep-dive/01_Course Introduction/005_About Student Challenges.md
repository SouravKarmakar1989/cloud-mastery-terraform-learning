# About Student Challenges

## Pointwise No-Loss Summary

1. Student challenges are provided at end of each section.
2. Purpose is hands-on practice of course concepts in learner's own AWS account.
3. Challenges are framed as manager-assigned task tickets.
4. Cost impact is intended to be minimal (small S3-file usage context).
5. Learners execute tasks independently first.
6. Solution videos are provided under each challenge for guidance.
7. Instructor strongly encourages completing challenges to reinforce practical skills.
