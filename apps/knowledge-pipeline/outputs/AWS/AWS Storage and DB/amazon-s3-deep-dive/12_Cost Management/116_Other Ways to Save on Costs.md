# Other Ways to Save on Costs

## Pointwise No-Loss Summary

1. Additional savings can come from architecture and placement decisions beyond storage class selection.

## Tactics Mentioned

1. Consider region selection/pricing differences where workload permits.
2. Re-evaluate replication scope so only necessary data is copied across regions/accounts.
3. Use AWS-native query/analytics services against in-place S3 data to reduce unnecessary egress/download costs.

## Outcome

1. Learner can pursue non-obvious cost reductions through data-placement, replication, and processing-strategy choices.
