# Tools That Can Help with Cost Management

## Pointwise No-Loss Summary

1. AWS provides multiple native tools for S3 cost visibility and optimization.
2. Different tools serve different depth levels from high-level trends to granular request analysis.

## Native Tools Discussed

1. Cost Explorer:
   - service-level cost trend analysis and filtering.
2. S3 Storage Lens:
   - S3-focused usage/activity metrics and optimization insights.
3. Cost and Usage Report (CUR):
   - raw detailed billing data for advanced analytics.
4. CloudWatch S3 metrics:
   - operational metrics tied to request/size behavior.
5. S3/CloudTrail logging data:
   - granular event/request visibility for deeper investigation.
6. S3 Inventory:
   - object-level catalogs supporting cost hygiene analysis.

## Practical Guidance

1. Combine tools rather than relying on one dashboard.
2. Use native signals first, then optional third-party analytics platforms when deeper cross-domain FinOps workflows are needed.

## Outcome

1. Learner has a clear tool-selection map for ongoing S3 cost governance.
