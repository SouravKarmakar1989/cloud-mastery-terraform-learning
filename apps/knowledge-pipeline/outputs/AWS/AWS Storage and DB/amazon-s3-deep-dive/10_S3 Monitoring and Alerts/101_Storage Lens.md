# Storage Lens

## Pointwise No-Loss Summary

1. S3 Storage Lens provides centralized visibility into S3 storage usage and activity metrics.
2. It supports analysis at multiple scopes (for example organization/account/bucket/prefix depending on tier).

## What It Helps With

1. Storage usage understanding.
2. Cost-optimization opportunity identification.
3. Data-management recommendations.
4. Trend analysis over retained metric periods.

## Free vs Advanced Positioning

1. Free tier:
   - baseline usage metrics,
   - shorter retention,
   - bucket-level granularity.
2. Advanced tier:
   - additional activity metrics,
   - longer retention,
   - prefix-level granularity,
   - recommendations and richer analytics features.

## Operational Use Cases Mentioned

1. Detect low-usage data candidates for lifecycle transitions.
2. Attribute storage behavior to teams using finer granularity.
3. Analyze long-term growth trends.
4. Spot month-over-month outliers/anomalies.

## Export Capability

1. Data can be exported for external analysis/custom dashboards and integration workflows.

## Outcome

1. Learner understands Storage Lens as primary built-in analytics dashboard for S3 estate optimization.
