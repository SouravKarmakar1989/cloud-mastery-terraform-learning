Welcome to the section on Agentic AI. One of the most exciting and fast moving areas in modern generative AI development. In this part of the course, we focus on building systems that don't just generate responses but can reason, plan, take action, collaborate, and integrate with real tools and data across your organization. We'll start by introducing LM agents in Amazon Bedrock, and you'll get hands on experience creating your first bedrock agent. From there, we'll explore multi-agent workflows, how agents coordinate with one another, and how they use short and long term memory to maintain context and improve task performance. We'll dive deep into AWS native Agentic ecosystem, including strands agents, AWS Agent Squad, and Agent Core, and you'll walk through a lab that combines Strands agents with Agent Squad to orchestrate more advanced behaviors. You'll also learn how to extend agents with external tools and capabilities using the Model Context Protocol, or MCP. We'll cover how to deploy MCP servers on ECS or Lambda, how to define tools with open API, and how to integrate humans into the loop for review, guidance and oversight. Then we move into the world of Amazon Q, starting with Amazon Q business, where you'll build an agent that can securely interact with your company's documents, apps, and knowledge sources. We'll go hands on with Q business and Q apps, learn how to design enterprise grade workflows, and clean up resources properly when you're done. Finally, we'll cover Amazon Q developer, a powerful tool for accelerating your own development process. You'll practice using it hands on as a coding partner, debugger, and generator of integration scaffolding for your AI applications. By the end of this section, you'll know how to design, deploy, and orchestrate fully featured Agentic systems across Bedrock strands Agent Squad, MCP, and Amazon Q. Let's get started.
