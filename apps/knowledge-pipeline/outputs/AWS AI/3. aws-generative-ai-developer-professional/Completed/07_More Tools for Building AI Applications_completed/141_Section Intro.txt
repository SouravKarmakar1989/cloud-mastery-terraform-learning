Welcome to more tools for building generative AI applications. A deep dive into the AWS services and architectural patterns that let you turn Foundation models into fully featured, production ready applications. In this section, you'll learn how to connect your AI logic to the rest of your system, automate workflows, and build the integration points that make your application scalable, reliable, and enterprise grade. We'll start with AWS Lambda and walk through multiple integration patterns, including invoking bedrock directly from serverless functions. Then we'll explore Amazon API gateway and how to expose gen AI functionality securely through custom APIs. You'll also learn how to manage dynamic settings like on the fly model selection using AWS Appconfig. From there, we'll move into AWS Step Functions, where you'll learn how to design state machines, manage stopping conditions, apply circuit breaker patterns, and implement complex agentic reasoning workflows like react and chain of thought routing. You'll also build a hands on project that chains prompts together using step functions and bedrock. We'll then look at the DevOps side of gen AI development using code, build, code, deploy, and code pipeline to create automated CI CD workflows. You'll also explore MLflow as a tool for experimentation and tracking performance across different LM configurations. Next, we'll cover a variety of services that expand the capabilities of your AI applications. Appsync for GraphQL APIs, outposts, and wavelength for hybrid and edge deployments. Amazon SQS for asynchronous workloads. AWS amplify for rapid front end integration and event driven patterns with EventBridge, SNS, and Appflow. We'll wrap up with best practices for creating Foundation model API interfaces, ensuring your generative AI components are accessible, secure, and easy to integrate across your organization. By the end of this section, you'll be comfortable building full gen AI powered applications using the broader AWS ecosystem, connecting models to tools, workflows, and infrastructure that work at scale. Let's get started.
